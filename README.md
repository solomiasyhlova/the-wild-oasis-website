<h1><b>The Wild Oasis</b></h1>

The Wild Oasis is a modern hotel booking web application that allows users to explore luxury cabins, make reservations, and manage their bookings through a clean and intuitive interface. The application provides a seamless booking experience with real-time availability, secure authentication, and a fully responsive design.

<b>Features</b>

- Browse available cabins with detailed information, pricing, and images
- Check cabin availability and make reservations
- Authentication with Google Sign-In
- Manage personal bookings and reservations
- Responsive design optimized for desktop, tablet, and mobile devices
- Synchronization powered by Supabase
- Dynamic routing and server-side rendering with Next.js

<b>Tech Stack</b>

- Next.js 
- React 
- Supabase 
- Tailwind CSS
- React Context API
- React Hook Form 
- Vercel
